## ðŸŽ¨ Customization Guide

You can easily adjust the loader's appearance using the following CSS variables:

- `--loader-primary`: Adjusts the primary color of the loader. Use your theme's `--primary` variable.
- `--loader-secondary`: Controls the secondary color, usually used for backgrounds or accents.
- `--loader-border`: Sets the border thickness for the loader rings.
- `--loader-width`: Defines the overall size of the loader.
- `--loader-speed`: Controls the speed of the animation.

### Example Customization

```css
:root {
  --loader-primary: hsl(220, 90%, 56%);
  --loader-secondary: hsl(220, 20%, 80%);
  --loader-border: 3px;
  --loader-width: 64px;
  --loader-speed: 1.5s;
}
```

## Usage Instructions

1. Copy the CSS and place it in your stylesheet (global.css or relevant module
   file).
2. Use the HTML structure in your component or .mdx file.
3. Customize the variables (`--primary`, `--secondary`) to fit your theme.
