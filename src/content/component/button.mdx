## Features

1.  **Variant-based Styling**:
    - Five distinct variants: `contained`, `outlined`, `ghost`, `text`, and `link`.
2.  **Color Options**:
    - Supports `primary`, `success`, and `destructive` colors for buttons.
3.  **Size Variants**:
    - Sizes include `tiny`, `sm`, `lg`, and `icon` for compact or prominent designs.
4.  **Loading State**:
    - Built-in loading support with customizable indicators and positions (`start` or `end`).
5.  **Customizable**:
    - Supports icons (`startIcon`, `endIcon`) and animation options (`disableAnimation`).

## Variants

### Contained

The `Contained` button variant is ideal for primary actions and tasks that require the most emphasis. It features a solid background with clear visual prominence, making it suitable for actions like "Submit," "Buy Now," or "Get Started."

The `Outlined` button variant provides a lightweight alternative to the Contained button. It uses a border and no solid background, making it less visually heavy, yet suitable for secondary actions or less critical tasks like "Cancel" or "More Info."

### Text

The `Text` button variant removes all background, border, and hover effects except for the text itself. It appears like plain text but maintains button interactivity. On hover, only the text color changes subtly, offering a minimalist yet functional design. This is useful for inline actions or tertiary interactions like "Read More" links.

### Link

The `Link` button variant is specifically designed to mimic the behavior of traditional hyperlinks while maintaining the styling and accessibility of a button. It is perfect for scenarios where navigation or linking to other pages is required.

Integration with routing libraries like `Next.js` using the `asChild` prop to wrap a `Link` component, ensuring seamless client-side navigation with consistent styling.

### Add the Component Code

Copy the full component code and place it in your project under the desired directory, for example: `components/lib/button.tsx`.

```tsx
import * as React from "react";
import { Slot, Slottable } from "@radix-ui/react-slot";
import { cva, type VariantProps } from "class-variance-authority";

import { cn } from "@/lib/utils";

const buttonVariants = cva(
  "inline-flex shrink-0 cursor-pointer select-none flex-wrap items-center justify-center text-center leading-[1em] gap-2 font-medium no-underline duration-[0.2s] ease-out transition-[color,background-color,border-color,opacity,box-shadow,transform] shadow border-transparent border-transparent focus-visible:outline focus-visible:outline-2 focus-visible:outline-offset-2 motion-reduce:hidden  active:scale-95 disabled:cursor-not-allowed disabled:scale-100 disabled:bg-muted disabled:text-muted-foreground [&_svg]:pointer-events-none [&_svg]:shrink-0",
  {
    variants: {
      variant: {
        contained: "bg-background text-foreground hover:bg-foreground/5",
        outlined: "border border-border shadow-none",
        ghost:
          "text-foreground shadow-none hover:bg-accent hover:text-foreground",
        text: "shadow-none cursor-auto",
        link: "shadow-none underline-offset-4 hover:underline",
      },
      color: {
        default: "focus-visible:outline-foreground",
        primary: "focus-visible:outline-primary",
        success: "focus-visible:outline-success",
        destructive: "focus-visible:outline-destructive",
      },
      size: {
        tiny: "h-5 px-2 py-0 text-xs [&_svg]:size-3 gap-1",
        sm: "h-7 px-3 text-xs [&_svg]:size-3 gap-1.5",
        md: "h-9 px-4 text-sm [&_svg]:size-4 ",
        lg: "h-10 px-8 text-lg [&_svg]:size-5",

        none: "rounded-none",
        sm: "rounded-sm",
        md: "rounded-md",
        lg: "rounded-lg",

      },
    },

      variant: "contained",
      size: "md",
      color: "default",
      rounded: "lg",
    },

      // Contained
      {
        variant: "contained",
        color: "primary",
        className: "bg-primary text-primary-foreground hover:bg-primary/80",

const defaultLoadingIndicator = (
  <svg
    className="animate-spin"
    xmlns="http://www.w3.org/2000/svg"
    fill="none"
    viewBox="0 0 24 24"
  >
    <circle
      className="opacity-25"

export interface ButtonProps
  extends Omit<React.ButtonHTMLAttributes<HTMLButtonElement>, "color">,
    VariantProps<typeof buttonVariants> {
  asChild?: boolean;
  disableAnimation?: boolean;
  noPadding?: boolean;
  startIcon?: React.ReactNode;
  endIcon?: React.ReactNode;
  loading?: boolean;
  loadingPosition?: "start" | "end";
  loadingIndicator?: React.ReactNode;
}

const Button = React.forwardRef<HTMLButtonElement, ButtonProps>(
  (
    {
      className = "",
      variant,
      disableAnimation,
      noPadding,
      size,
      color,
      asChild = false,
      startIcon,
      endIcon,
      disabled,
      rounded,
      loading = false,
      loadingPosition = "start",
      loadingIndicator = defaultLoadingIndicator,
      children,
      ...props
    },
    ref,
  ) => {
    const Comp = asChild ? Slot : "button";

    const renderStartIcon =
      loading && loadingPosition === "start"
        ? loadingIndicator
        : (startIcon ?? null);

    const renderEndIcon =
      loading && loadingPosition === "end"
        ? loadingIndicator
        : (endIcon ?? null);

    return (
      <Comp
        className={cn(
          buttonVariants({ variant, size, color, rounded }),
          disableAnimation && "active:scale-0",
          noPadding && "h-auto p-0",
          className,
        )}
        ref={ref}
        aria-busy={loading}
        aria-label={props["aria-label"] || (loading ? "Loading" : undefined)}
        disabled={loading || disabled}
        {...props}
      >
        {renderStartIcon && renderStartIcon}
        <Slottable>{children}</Slottable>
        {renderEndIcon && renderEndIcon}
      </Comp>
    );
  },
);
Button.displayName = "Button";

export { Button, buttonVariants };
```

## API Reference

### ðŸŽ¨ Props

The variant prop controls the visual appearance of the button:

| **Name**           | **Type**                                              | **Default**         | **Description**                                                                                                   |
| ------------------ | ----------------------------------------------------- | ------------------- | ----------------------------------------------------------------------------------------------------------------- |
| `variant`          | `"contained", "outlined", "ghost", "text", "link"` | `"contained"`       | Defines the visual style of the button.                                                                           |
| `color`            | `"default", "primary", "success", "destructive"`   | `"default"`         | Specifies the color scheme.                                                                                       |
| `size`             | `"tiny", "sm", "md", "lg", "icon"`                | `"md"`              | Determines the size and padding of the button.                                                                    |
| `rounded`          | `"none", "sm", "md", "lg", "full"`                | `"lg"`              | Controls the border-radius of the button. Enables customization of button corner styles for various design needs. |
| `asChild`          | `boolean`                                             | `false`             | If `true`, renders the button as a child of another component using the `Slot` component.                         |
| `disableAnimation` | `boolean`                                             | `false`             | Disables the animation effects when set to `true`.                                                                |
| `noPadding`        | `boolean`                                             | `false`             | Removes all padding from the button when set to `true`.                                                           |
| `startIcon`        | `React.ReactNode`                                     | `null`              | Icon or element to render at the start of the button.                                                             |
| `endIcon`          | `React.ReactNode`                                     | `null`              | Icon or element to render at the end of the button.                                                               |
| `loading`          | `boolean`                                             | `false`             | Shows a loading indicator when set to `true`.                                                                     |
| `loadingPosition`  | `"start", "end"`                                     | `"start"`           | Determines whether the loading indicator appears at the start or end of the button.                               |
| `loadingIndicator` | `React.ReactNode`                                     | Default spinner SVG | Custom loading indicator to display when `loading` is `true`.                                                     |
| `disabled`         | `boolean`                                             | `false`             | Disables the button when set to `true`.                                                                           |
